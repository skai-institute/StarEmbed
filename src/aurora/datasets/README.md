# StarEmbed datasets

Here, you can find descriptions of all StarEmbed datasets and Python code for converting them to the standardized StarEmbed schema in HuggingFace datasets format. Each dataset has their own directory beneath `datasets/`. We also include a script to validate the HuggingFace dataset output by the conversion script and to provide high level statistics for the variable star classes included in each dataset.
