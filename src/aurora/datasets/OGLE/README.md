# OGLE

This dataset is the OGLE4 Collection of Variable Stars ([Udalski, et al. 2015](https://acta.astrouw.edu.pl/Vol65/n1/a_65_1_1.html)).



### Download raw data
The entire dataset can be explored [here](https://www.astrouw.edu.pl/ogle/ogle4/OCVS/) and downloaded in chunks or downloaded all at once:

```bash
wget -r -nH -P {output directory here} ftp://ftp.astrouw.edu.pl/ogle/ogle4/OCVS/
```
