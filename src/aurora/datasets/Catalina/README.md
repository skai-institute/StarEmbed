# Catalina Surveys

This dataset is the Catalina Northern Periodic Variable Star Catalog ([Drake, et al. 2014](http://arxiv.org/abs/1405.4290)).



### Download raw data

```bash
curl -o CSDR1_varstars_v2.txt http://nesssi.cacr.caltech.edu/DataRelease/CatalinaVars.tbl
curl -o Catalina_PVars_DR2.phot http://nesssi.cacr.caltech.edu/DataRelease/AllVar.phot.gz
```
