## ZTF 40K Dataset

The script to cross-match the ZTF and CSDR1 stars is at `/datasets/ZTF/query_csdr1_from_ztf.py`.

The script to build the train-val-test split is at `/datasets/ZTF/build_split.py`, we also upload the train-val-test split onto `https://huggingface.co/datasets/123anonymous123/StarEmbed/tree/main/data`.
